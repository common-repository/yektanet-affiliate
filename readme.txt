Increase your sales with Chavosh Affiliate Marketing System. This plugin helps you start your affiliate marketing with Chavosh.

== Description ==

This is the official Chavosh Affiliate Marketing System plugin for WooCommerce that connects your WooCommerce website to Chavosh. With this plugin, you send your purchase data to our server so that we can calculate your share of it.

== Installation ==

First install and activate the plugin and then put your token and app_id at the settings page.
